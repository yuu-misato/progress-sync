# Progress Sync (進捗管理システム)

製造業・制作現場向けの、直感的でプレミアムな進捗管理システムです。
ヤマト運輸の配送追跡システムのような視認性の高さと、日本的なビジネスシーンにマッチしたデザインを両立しています。

## 特徴
- **お客様向け画面**: ログイン不要で、プロジェクトの現在地が一目でわかる「タイムライン追跡UI」。
- **制作者向け画面**: 期限管理、タスクの可視化、提出アクションへの導線。
- **デザイン**: 信頼感のあるブルーを基調とした、モダンで洗練されたGlassmorphismデザイン。

## 起動方法

```bash
npm install
npm run dev
```

## ページ構成

- **トップページ**: `http://localhost:3000/`
  - 入口の選択（お客様 / 制作者）
- **お客様デモ画面**: `http://localhost:3000/client/demo-project`
  - 進捗トラッカー（要件定義 -> デザイン -> 実装...）
  - アクティビティログ
- **制作者ダッシュボード**: `http://localhost:3000/worker`
  - タスク一覧
  - 期限リマインド
  - 統計情報
